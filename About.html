MILES - Version 1.0

Minimized Linear Inequalities for Large S-boxes

MILES generates minimized linear inequalities of a given S-box using Espresso* algorithm.
MILES consists of four processes:
1. S-box to DDT Generation
2. DDT to Truth Table Conversion
3. Truth Table Minimization
4. Linear Inequalities Generation
(* - https://embedded.eecs.berkeley.edu/pubs/downloads/espresso/)
Currently MILES is available for Linux Operating System only.
